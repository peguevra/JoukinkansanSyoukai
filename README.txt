# 常勤換算表 自動作成システム

勤務表CSVから常勤換算表を自動作成するツールです。

## デモ動画

GitHub Pages

https://peguevra.github.io/JoukinkansanSyoukai/

## 概要

毎月の常勤換算表作成において、

* 勤務実績の集計
* 転記作業
* 計算確認

などに時間を要していませんか。

本ツールは勤務表CSVを入力することで、常勤換算表を自動作成します。

## 特徴

* 勤務表CSVから自動集計
* 帳票への転記を自動化
* 入力ミス・転記ミスを削減
* 属人化の解消
* 数クリックで帳票完成

## デモ内容

デモ動画では、

1. 入力データ（勤務表CSV）
2. ツール実行
3. 常勤換算表出力

までの一連の流れをご覧いただけます。
